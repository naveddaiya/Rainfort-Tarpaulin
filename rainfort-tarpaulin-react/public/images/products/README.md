## üìù How to Add Images

1. **Save your product photo** to this folder
2. **Open** `src/data/products.js`
3. **Reference the image** in your product:
   ```javascript
   image: "/images/products/your-image-name.jpg"
   ```

## üé® Image Tips

### For Best Results:
- Use a neutral background (white, light gray)
- Show the product clearly and prominently
- Ensure good lighting (avoid shadows)
- Keep the same aspect ratio for all images
- Show the product in use when possible

### Image Editing:
- Crop to 800x600 pixels
- Adjust brightness/contrast if needed
- Compress using tools like:
  - [TinyPNG](https://tinypng.com/)
  - [Squoosh](https://squoosh.app/)
  - [ImageOptim](https://imageoptim.com/)
